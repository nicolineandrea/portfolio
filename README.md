# Nicoline Andrea Janka Portfolio

Static portfolio website.

## Structure

- `index.html` is the GitHub Pages entry point.
- Main subpages:
  - `about-me.html` - About Me
  - `bands-live.html` - Bands & Live
  - `audiovisual-works.html` - Audiovisual Works
  - `sound-digital-works.html` - Sound & Digital Works
  - `contact-and-press.html` - Contact & Press
- `assets/css/` contains stylesheets.
- `assets/js/` contains JavaScript.
- `assets/images/` contains optimized site images grouped by project area.
- `assets/videos/` contains site-ready video files.
- `assets/raw-video/` contains large source footage and is ignored by Git.

## Local Preview

Run a local server from this folder:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```
